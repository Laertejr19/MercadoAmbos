# Mercado Ambos
Laerte, Laerte, Guilherme e Sarah

## Problema

O cliente é um comerciante local, dono de um mini mercado em Sapucaia do Sul. Através de contatos, o mesmo demonstrou dificuldade em organizar suas financias, sendo estes a compra "fiada" de clientes e o pagamento para fornecedores, devido ao método manual que utiliza. 

## Solução

A equipe realizou a criação de um website voltado para suprir e facilitar as necessidades do cliente, utilizando a ferramenta de criação de contas tanto para fornecedores quanto para clientes, podendo editar e excluir as mesmas. Além disto, o próprio website realiza a atualização dos valores cobrados com base no juros inserido na hora da criação assim que realizado o pagamento.

## Ferramentas Utilizadas

- HTML
- CSS
- JavaScript
- FireBase

## Como utilizar

- Clone o repositório ou baixe os arquivos.
- Abra o arquivo index.html em seu navegador.
- Selecione o botão de cadastrar.
- Registre a conta do seu fornecedor ou de seu cliente.
- Edite ou exclua caso desejar atráves dos botões.
- Tenha acesso as contas separadas pelo botão "Acessar".
